3-Layer Architecture

Authentication & Security

Token-Based Authentication

Process:

  • User token ‚Üí Session token
  • 2-minute token lifetime
  • Auto-refresh at 30 seconds
  • Retry with exponential backoff

Benefits:

  • Seamless user experience
  • Efficient token reuse
  • Resilient to network issues
  • Automatic recovery

Key Features

5 Core MCP Tools

1. ping() - Health check and connectivity test
2. list_namespaces() - Browse tag hierarchy and folder structure
3. search_tags(pattern) - Find tags using wildcard patterns
4. get_tag_metadata(tag_path) - Retrieve tag properties, units, ranges
5. read_timeseries(tags, start, end) - Fetch historical data with natural language time support

Natural Language Time Parsing

Converts Human Language ‚Üí ISO Timestamps

User Input System Output
"yesterday" 2025-10-30T00:00:00Z
"past 24 hours" Current time - 24 hours
"last week" Current time - 7 days
"now" Current timestamp
Key Benefits: Users don't need to know timestamp formats or timezones
